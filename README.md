## 📖 Course Overview
[cite_start]Communication Lab is a required course for the CSE program, designed to teach the basics of data communication through experimental works[cite: 65, 68]. [cite_start]Understanding and implementing key concepts of data communication and networking through 10 to 12 hands-on lab classes are especially important for students aiming to develop a career in networking[cite: 66]. 

### Objectives
* [cite_start]Build an understanding of the fundamental concepts of data communications and Networking[cite: 70].
* [cite_start]Familiarize the student with the basic taxonomy and terminologies of the Computer networking area[cite: 71].

## 🧪 Tentative Lecture & Experiment Plan

[cite_start]This repository contains the practical implementations for the following lab topics[cite: 95]:

| Lab | Topic | Details & Objectives |
| :---: | :--- | :--- |
| **01** | [cite_start]**Understanding IP & Network Topology - Lab 1** [cite: 96] | [cite_start]Ensure proper understandings of IP, NetID, HostID, SubnetMask, Default Gateway, etc[cite: 96]. |

| **04** | [cite_start]**Basic LAN** [cite: 96] | [cite_start]Design and configure a basic LAN[cite: 96]. |
| **05** | [cite_start]**LANs connected with a single router** [cite: 96] | [cite_start]Design and configure LANs connected with a single router[cite: 96]. |

| **07** | [cite_start]**Advanced MAN** [cite: 100] | [cite_start]Design an advanced MAN and configure the routing[cite: 100]. |
| **08** | [cite_start]**WAN** [cite: 100] | [cite_start]Design a WAN and explain the configuration needed for this setup[cite: 100]. |
| **09** | [cite_start]**Modulation** [cite: 100] | [cite_start]Converting Analog to Digital Signals[cite: 100]. |
| **10** | [cite_start]**AM & FM** [cite: 100] | [cite_start]Implement Amplitude Modulation and Frequency Modulation[cite: 100]. |
