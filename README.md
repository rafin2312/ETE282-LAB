## 📖 Course Overview
Communication Lab is a required course for the CSE program[cite: 65]. [cite_start]It focuses on the understanding and implementation of key concepts of data communication and networking through hands-on lab classes. This repository contains scripts, simulations, and network topology models designed to bridge the gap between theoretical concepts and practical implementation.

### Objectives
* Build an understanding of the fundamental concepts of data communications and Networking.
* Familiarize the student with the basic taxonomy and terminologies of the Computer networking area.

---

## 🛠️ Tools & Technologies Used
This repository utilizes the following tools to complete the lab experiments:
* **Networking Simulation:** Cisco Packet Tracer (for building and analyzing LAN/MAN/WAN topologies).
* **Programming Languages:** Python, C, and C++ (for writing modulation algorithms, signal processing scripts, and multiplexing logic).

## 🧪 Experiments Covered

Here is the list of core lab experiments implemented in this repository based on the syllabus:

| Lab | Topic | Objectives |
| :---: | :--- | :--- |
| **01 & 02** | **IP & Network Topology** | Understand IP, NetID, HostID, SubnetMask, and Default Gateway. |
| **03** | **Understanding Mediums** | Learn which mediums to use for connections; create copper cross-over and straight-through cables; understand Console and serial ports. |
| **04** | **Basic LAN** |Design and configure a basic LAN. |
| **05** | **Router Connections** | Design and configure LANs connected with a single router. |


[Image of network topology diagram]

| **06 & 07** | **MAN & Advanced MAN** | Design a MAN and apply appropriate routing for this topology. |
| **08** | **WAN** | Design a WAN and explain the required configuration. |
| **09** | **Modulation** | Convert Analog to Digital Signals. |
| **10** | **AM & FM** | Implement Amplitude Modulation and Frequency Modulation. |


[Image of amplitude and frequency modulation signals]

| **11** | **CDMA** | Understand Multiplexing in the form of Code Division Multiple Access. |
