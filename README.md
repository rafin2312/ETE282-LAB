## 📖 About the Project

In modern telecommunications, understanding how signals are modulated, encoded, and transmitted across various channels is crucial. This repository contains laboratory experiments that analyze time-domain and frequency-domain characteristics of signals, multiplexing techniques, and the impact of Additive White Gaussian Noise (AWGN) on data integrity.

## 🛠️ Tech Stack

* **Programming Languages:** Python 3.x, C, C++
* **Data Science & Visualization:** `NumPy`, `SciPy`, `Matplotlib`
* **Networking Simulation:** Cisco Packet Tracer

## 🧪 List of Experiments

| Exp No. | Topic | Description | Language/Tool |
| :---: | :--- | :--- | :--- |
| **01** | **Analog Modulation** | Amplitude (AM) and Frequency (FM) Modulation & Demodulation. | Python |
| **02** | **Pulse Code Modulation** | Sampling theorem validation, quantization, and PCM encoding. | Python / C++ |
| **03** | **Digital Modulation** | ASK, FSK, and PSK implementation and spectral analysis. | Python |
| **04** | **Line Coding Schemes** | Unipolar, Polar, Bipolar (AMI), and Manchester coding. | C / C++ |
| **05** | **Channel Noise & BER** | Simulating AWGN channels and evaluating Bit Error Rate (BER). | Python |
| **06** | **Network Topologies** | Data transmission simulations across configured network nodes. | Packet Tracer |
